# Análise de Emendas Parlamentares

## Descrição
Análise de um dataset de emendas parlamentares utilizando Python e pandas.

## O que foi feito
- Carregamento do dataset e visualização das 10 primeiras linhas
- Identificação da quantidade de linhas e colunas
- Análise das colunas e seus significados
- Verificação de valores nulos
- Análise dos tipos de dados das variáveis
- Análise exploratória de dados (EDA)
- Identificação do ano mais recente
- Verificação de duplicados
- Análise de completude dos dados

## Ferramentas
- Python
- Pandas
- Google Colab
